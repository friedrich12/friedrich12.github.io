Operating Systems Concepts: Paging

Posted at — Jan 1, 0001

Why do we need paging? In operating systems paging is used to elimiate the need for contigous physical memory. By using virtual memory with paging we can use blocks in the computer’s physical memory instead of using a contigous section. But still why does this matter?

Just look at figure above. If we only use contingous blocks of memory we might be taking away memory that can be used to run other programs. For example, if we allocate 4MB of memory for a text editor but later we find that our text editor is only using 2MB of memory we need to free the memory it is not using. Without paging the only way to free the memeory is to kill the whole program, which is not what we want to do.

// Example here

With paging we can allocate blocks of memory and place them in different sections in the computer’s physical memory, thus, the memory need not be contigous.
